🔥 WHAT EACH FILE DOES
═════════════════════════════════════════════════════════════════════════════════

setup.py
  → Automatically creates project structure
  → Creates all directories (model/, utils/, server/, clients/, dashboard/)
  → Generates all Python source files
  → One-command initialization

requirements.txt
  → Lists all Python packages needed
  → Install with: pip install -r requirements.txt
  → Includes: Flower, PyTorch, scikit-learn, Streamlit

server/server.py
  → Runs the Federated Learning Server
  → Waits for all 3 clients
  → Aggregates model weights using FedAvg
  → Evaluates global model accuracy

clients/client.py
  → Federated Learning Client (runs on each "hospital")
  → Trains model on local data
  → Communicates with server
  → Sends/receives weights

model/model.py
  → Defines SimpleNN neural network
  → 30 inputs → 128 → 64 → 2 outputs
  → Used by both server and clients

utils/data_loader.py
  → Loads Breast Cancer dataset
  → Splits data across 3 clients
  → Handles preprocessing

utils/training.py
  → train_epoch() - trains for N epochs
  → evaluate() - computes accuracy & loss
  → Core training logic

dashboard/app.py
  → Streamlit monitoring dashboard
  → Shows connected clients
  → Displays training progress
  → Real-time metrics
