================================================================================
⚙️ CUSTOMIZATION EXAMPLES
================================================================================

Want to change something? Here are common modifications:

INCREASE TRAINING ROUNDS:
  Edit: server/server.py
  Change: start_server(num_clients=3, num_rounds=10)
  Effect: More iterations = potentially better accuracy

CHANGE LEARNING RATE:
  Edit: clients/client.py
  Change: lr=0.0001 (smaller = slower learning)
  Effect: Affects convergence speed

MODIFY MODEL SIZE:
  Edit: model/model.py
  Change: self.fc2 = nn.Linear(128, 32)
  Effect: Smaller model = faster training but lower accuracy

USE DIFFERENT PORT:
  Edit: server/server.py & clients/client.py
  Change: server_address="0.0.0.0:8090"
  Effect: Useful if port 8080 is in use

================================================================================
🔧 TROUBLESHOOTING QUICK REFERENCE
================================================================================

PROBLEM: Import Error (flwr not found)
  SOLUTION: pip install -r requirements.txt

PROBLEM: Port 8080 already in use
  SOLUTION: Change port to 8090 in both server.py and client.py

PROBLEM: Clients won't connect
  SOLUTION: Start server first, wait for "Waiting for clients"

PROBLEM: Out of memory
  SOLUTION: Reduce batch_size in utils/data_loader.py

PROBLEM: Very slow training
  SOLUTION: Normal on CPU. Consider GPU or reduce model size
